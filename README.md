# LeoDev - Portafolio

Portafolio profesional de desarrollador frontend. Sitio web personal para mostrar proyectos, habilidades y formas de contacto.

## Tecnologías

- HTML5
- CSS3 (Custom Properties, Flexbox, Grid, Animaciones)
- JavaScript (ES6+ Modules, IntersectionObserver)
- Google Fonts (Inter)
- GitHub Pages

## Secciones

- **Inicio** — Presentación con efecto de máquina de escribir
- **Sobre mí** — Breve introducción personal
- **Habilidades** — Tecnologías y herramientas de trabajo
- **Proyectos** — Galería de proyectos con cards dinámicas
- **Footer** — Redes sociales (LinkedIn, Instagram, GitHub)

## Contacto

- WhatsApp integrado como botón flotante
- Enlaces a LinkedIn e Instagram en el footer
